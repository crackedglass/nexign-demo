# Nexign Demo Service

## Запуск
### Используя Maven
#### Linux / MacOS
```bash
git clone https://github.com/crackedglass/nexign-demo.git
cd nexign-demo
mvn install
cd impl
mvn spring-boot:run
```
#### Windows
```bat
git clone https://github.com/crackedglass/nexign-demo.git
cd nexign-demo/impl
.\m
```
## Используемые библиотеки и инструменты:

- Spring Boot 3
- Spring Web
- Java 17
- Maven
- H2
- Liquibase
- JOOQ
- Swagger
